# Президентское расписание: бот

## Установка
Бот построен на `aiogram 3` и `aiogram_dialog`. 
Для установки используйте: 
```
pip install -r requirements.txt
python3 main.py
```

## Описание модулей
`presidential_timetable.client`: клиент взаимодействия с REST API бэкенда  
`presidential_timetable.core`: модели всякие  
`presidential_timetable.bot`: логика самого бота
